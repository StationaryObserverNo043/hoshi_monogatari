🌌 星ヲ巡る旅
〜アナタのメモで紡ぐ物語〜
地球侵略阻止ミッション が贈る、ブラウザで動くSFログアプリデス。
アナタの日々の記録が「星を灯し」、アナタだけの星座と物語を紡いでいきマス。
7つの記録で1章が完成し、新たな銀河へと旅立ちマス🛸
🚀 デモ
▶ 星ヲ巡る旅を起動する
✨ 主な機能
📝 メモログ — テキストを入力して星を灯す。7件で1章クリア
🌠 星空Canvas — 記録が星として宇宙に配置され、星座ラインで繋がる
⚡ 引力エフェクト — マウス/タッチで星が引き寄せられる
🔭 星座生成＆Xシェア — 章完了時に唯一無二の星座画像を生成してポスト可能
💾 ローカル保存 — 外部サーバー不要。記録はブラウザに保存
📦 エクスポート/インポート — JSONファイルで旅の記録を持ち運べる
🛸 使い方
HTMLファイルをブラウザで開く（またはGitHub Pagesにアクセス）
「▶ ミッション開始」でガイドを確認
「🚀 宇宙へ出発する」でアプリ起動
テキストを入力して「🌟 星を灯す」
~~テキスト~~ で取り消し線（本音や過去の自分へのツッコミに）
PC環境なら Ctrl + Enter で素早く送信
📁 ファイル構成
コード
外部APIへの通信は一切行いません。
ブラウザ単体で完結するため、ダウンロードしてオフラインでも動作シマス。
🧬 技術スタック
Vanilla HTML / CSS / JavaScript（フレームワーク不使用）
Canvas API（星空描画・引力エフェクト・星座生成）
LocalStorage（データ永続化）
Google Fonts（Orbitron / Noto Sans JP）
